# Ameet Web/H5 MVP

Mobile-first static frontend prototype for a guided relationship story product.

## Run locally

```bash
python3 -m http.server 4173
```

Open `http://127.0.0.1:4173`.

Demo invite code: `111111`

## Prototype routes

- `/#/`
- `/#/characters`
- `/#/characters/mina`
- `/#/chat/mina`
- `/#/account`

Replace `mina` with `chloe`, `ava`, `vanessa`, `riley`, or `elena`.

All character, story, choice, memory, and paywall content lives in
`src/data.js`.
